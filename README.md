# 🚀 Project‑1: AI Bot Creations  

A collection of experimental and production‑ready bots built with different approaches — from simple static responses to advanced AI integrations. This repo demonstrates how conversational agents can be designed using multiple paradigms: **static**, **API‑driven**, **AI‑powered**, and **RAG (Retrieval‑Augmented Generation)**.  

---

## 📂 Project Types
- **Static Bots** → Rule‑based responses, lightweight, no external dependencies.  
- **API Bots** → Connect to external services (REST/GraphQL) for dynamic answers.  
- **AI Bots** → Powered by LLMs (Gemini, GPT, LLaMA, etc.) with natural language understanding.  
- **RAG Bots** → Combine LLMs with custom knowledge bases for contextual, accurate replies.  

---

## 🛠️ Tech Stack
- **Languages**: Python, JavaScript  
- **Frameworks**: FastAPI, Flask, Streamlit  
- **AI Models**: Gemini 3.5, LLaMA, OpenAI GPT  
- **Libraries**: Requests, BeautifulSoup, LangChain, HuggingFace Transformers  
- **Databases**: SQLite, PostgreSQL (for RAG indexing)  

---

## ⚡ Features
- Modular design → Each bot type lives in its own folder.  
- Easy deployment → Run locally or host on cloud platforms.  
- Extensible → Add new bot types with minimal boilerplate.  
- Demo dashboards → Streamlit/React frontends for quick testing.
